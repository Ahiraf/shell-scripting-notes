# Labtest-Problems

This is my **CSE-336 (sessional)** labtest problem repository.
This repository contains solutions to some common lab test problems in **Bash scripting**.

---

## Problems Included

1. **Divisibility Check**
   - Checks if a number is divisible by 3, 5, or both.
   - Handles negative numbers as invalid input.
   - Script: `2204104_A.sh`

2. **Factorial Calculator**
   - Calculates the factorial of a given number.
   - Script: `2204104_B.sh`

3. **Recursive Power Calculator**
   - Calculates X raised to the power Y using recursion.
   - Accepts X and Y as command-line arguments.
   - Script: `2204104_C.sh`

4. **Color Variant**
   - Prints messages based on conditions for given color.
   - Script: `2204104_D.sh`
